KrakenBot Alpha v.0.1.10

KrakenBot is a specialized IRC bot meant to log a specific channel, 
and provide some interesting conversational and functional
abilities.

AUTHORS: Ethan Phillips, penguin1263, nsuarez19.

This project is based of the premade IRC bot Willie, and therefore, 
to avoid repercussions, we will now state that Willie is not our property,
nor do we take credit for it. More information about Willie can be found here:
https://github.com/embolalia/willie
